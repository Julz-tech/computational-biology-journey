# Computational Biology Journey

## About

This repository documents my journey toward becoming a better computational biologist.

I am learning how to integrate:

- Bioinformatics
- Data Science
- Machine Learning
- Statistics
- Artificial Intelligence

with the end goal of developing predictive models for biological systems.

## Current Focus

My current flagship project:

**Marine Heatwave Effects on Kelp Microbiome**

> Does maintaining an intact microbiome improve kelp resilience during marine heatwaves?
